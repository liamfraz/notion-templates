Common Questions

Can I import this into an existing Notion workspace?
Yes. Import the template as a new page in your existing workspace, then import each CSV file as a database. The setup guide walks you through every step.
Do I need a paid Notion plan?
The free Notion plan works perfectly. A paid plan gives you unlimited file uploads and additional collaboration features, but is not required.
Is the sample data realistic?
Yes. All sample data uses realistic Australian consulting scenarios with realistic hourly rates ($150-350/hr AUD), real-world project types, and practical client engagement examples.
What if I'm not in Australia?
Consultant OS works globally. Simply update the currency in the number columns to match your local currency. All workflows, templates, and structures are universal.
Can I customize the databases?
Absolutely. You can add columns, change select options, create new views, and modify any database to suit your workflow. The setup guide explains how the databases connect so you can extend them confidently.
