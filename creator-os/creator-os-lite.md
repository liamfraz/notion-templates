## Content Calendar

Plan every piece of content across every platform from one view.

### Database: Content Calendar

Import `content-calendar.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Title | Title |
| Platform | Select: YouTube / TikTok / Instagram / Twitter/X / LinkedIn / Newsletter / Blog / Podcast |
| Content Type | Select: Long-form Video / Short-form Video / Reel / Thread / Article / Newsletter Issue / Carousel / Podcast Episode |
| Status | Select: Idea / Scripting / Filming / Editing / Scheduled / Published / Evergreen |
| Publish Date | Date |
| Topic | Select: AI Tools Review / Productivity Tips / Behind the Scenes / Tutorial / Hot Take / Collaboration / Q&A |
| Category | Text |
| Engagement Goal | Text |
| Notes | Text |

### Content Lifecycle

```
Idea -> Scripting -> Filming -> Editing -> Scheduled -> Published -> Evergreen
```

### Recommended Views

- **Board View** — Kanban by Status (drag content through stages)
- **Calendar View** — Monthly calendar by Publish Date
- **By Platform** — Table grouped by Platform
- **This Week** — Table filtered to current week

### Batching Tip

Block your week: Monday for scripting, Tuesday for filming, Wednesday for editing, Thursday for scheduling. Friday for engagement and planning next week.

## Idea Bank

Never lose a content idea again. Capture everything, prioritise the best ones.

### Database: Ideas

Import `ideas.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Idea | Title |
| Platform | Select: YouTube / TikTok / Instagram / Twitter/X / LinkedIn / Newsletter / Blog / Podcast |
| Category | Text |
| Priority | Select: Hot / Medium / Low / Someday |
| Status | Select: Backlog / Researching / Ready to Create / In Production / Published |
| Source | Select: Audience Request / Trending / Evergreen / Collaboration / Personal |
| Potential | Select: High / Medium / Low |
| Notes | Text |

### Idea Priorities

| Priority | Action |
|---|---|
| **Hot** | Create this week — timely or highly requested |
| **Medium** | Schedule within 2 weeks |
| **Low** | Backlog — do when you have capacity |
| **Someday** | Review monthly, might never happen |

### Quick Score Before Creating

Ask yourself: Did someone ask for this? Can I monetise it? Can I repurpose it across 3+ platforms? If 2+ are "yes", move it to Ready to Create.

### Recommended Views

- **Board View** — Kanban by Priority
- **Ready to Create** — Filtered to Status = Ready to Create
- **By Platform** — Grouped by Platform

## Platform Manager

Track your presence across every platform in one place.

### Database: Platforms

Import `platforms.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Platform | Title |
| Handle | Text |
| Followers | Number |
| Monthly Growth | Number |
| Content Frequency | Text |
| Primary Format | Text |
| Status | Select: Active / Growing / Paused / New |
| Priority | Select: Primary / Secondary / Experimental |

### Platform Strategy

| Priority | Time Allocation |
|---|---|
| **Primary** | 60% of your effort — these drive growth and revenue |
| **Secondary** | 30% — repurpose primary content here |
| **Experimental** | 10% — test new platforms with low commitment |

### Cross-Platform Repurposing

Start with one long-form piece (YouTube video, blog post, podcast episode), then:
- Cut clips for TikTok and Reels
- Pull key points into a Twitter/X thread
- Create a carousel for Instagram
- Summarise in your newsletter
- Write a LinkedIn article with a professional angle

### Recommended Views

- **All Platforms** — Table sorted by Priority
- **Active Only** — Filtered to Active and Growing
