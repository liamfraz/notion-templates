## Income Streams

Track every revenue source and understand your profit margins.

### Database: Income Streams

Import `income-streams.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Stream Name | Title |
| Type | Select: Ad Revenue / Sponsorship / Affiliate / Digital Products / Coaching / Memberships / Merch / Licensing |
| Platform | Text |
| Monthly Revenue ($) | Number (currency format) |
| Monthly Expenses ($) | Number (currency format) |
| Net Profit ($) | Number (currency format) |
| Status | Select: Active / Growing / Paused / New / Declining |
| Effort (hrs/week) | Number |
| Started | Date |
| Notes | Text |

### Recommended Views

- **Dashboard** — Table sorted by Net Profit descending
- **By Type** — Grouped by Type to see revenue mix
- **Active Only** — Filtered to Active and Growing statuses
- **Effort vs Profit** — Table showing Effort and Net Profit side by side

### Quick Tip

If more than 50% of your income comes from one source, your business is fragile. Use this database to track your diversification and build towards having 4+ active income streams.

## Expenses

Track every business cost with tax categories.

### Database: Expenses

Import `expenses.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Expense | Title |
| Category | Select: Software / Equipment / Contractors / Advertising / Education / Travel / Office / Insurance / Accounting |
| Amount ($) | Number (currency format) |
| Frequency | Select: Monthly / Annual / One-off / Quarterly |
| Tax Deductible | Checkbox |
| Vendor | Text |
| Date | Date |
| Notes | Text |

### Recommended Views

- **All Expenses** — Table sorted by Date descending
- **By Category** — Grouped by Category
- **Tax Deductible** — Filtered to Tax Deductible = Yes
- **Recurring** — Filtered to Monthly and Annual items

### Quick Tip

Review your recurring expenses every quarter. Creators accumulate software subscriptions fast — if you have not used a tool in 30 days, cancel it.

## Goals

Set financial targets and track progress.

### Database: Goals

Import `goals.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Goal | Title |
| Category | Select: Revenue / Growth / Product / Brand Deals / Savings / Investment |
| Target ($) | Number (currency format) |
| Current ($) | Number (currency format) |
| Status | Select: On Track / Behind / Ahead / Completed / Not Started |
| Deadline | Date |
| Priority | Select: High / Medium / Low |
| Quarter | Select: Q1 2026 / Q2 2026 / Q3 2026 / Q4 2026 |
| Notes | Text |

### Recommended Views

- **Active Goals** — Filtered to exclude Completed
- **By Priority** — Kanban board by Priority
- **This Quarter** — Filtered to current quarter
- **Revenue Goals** — Filtered to Revenue category

### Quick Tip

Update goal progress at least monthly. If a goal has been "Behind" for two months straight, either adjust the target to something realistic or change your strategy — do not just ignore it.

## Monthly Finance Checklist

Run this on the 1st of every month. Takes 20 minutes.

### Income
- [ ] Log all income received last month
- [ ] Update each stream's monthly revenue
- [ ] Note any new or declining streams

### Expenses
- [ ] Log all business expenses
- [ ] Check for recurring charges you forgot
- [ ] Cancel any tools you have not used in 30 days

### Tax
- [ ] Transfer 25% of net profit to a separate savings account
- [ ] Save receipts for all deductible expenses

### Goals
- [ ] Update goal progress with current numbers
- [ ] Mark any completed goals
- [ ] Adjust unrealistic targets honestly
