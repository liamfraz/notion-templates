## Videos

Plan every video from idea to published. Track your production pipeline in one view.

### Database: Videos

Import `videos.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Title | Title |
| Status | Select: Idea / Scripting / Filming / Editing / Thumbnail / Scheduled / Published / Evergreen |
| Category | Select: Tutorial / Review / Vlog / Listicle / Commentary / Behind the Scenes / Collaboration / Shorts |
| Publish Date | Date |
| Target Length | Text |
| Thumbnail Status | Select: Not Started / Designing / A-B Testing / Final |
| SEO Score | Select: High / Medium / Low |
| Sponsor | Text |
| Notes | Text |

### Video Production Pipeline

```
Idea -> Scripting -> Filming -> Editing -> Thumbnail -> Scheduled -> Published -> Evergreen
```

### Recommended Views

- **Production Board** -- Board view grouped by Status (drag videos through stages)
- **Upload Calendar** -- Calendar view by Publish Date
- **By Category** -- Table grouped by Category
- **This Month** -- Table filtered to current month

### Publishing Tip

Aim for a consistent upload schedule. One well-produced video per week beats three rushed ones. The algorithm rewards consistency and watch time, not just volume.

## Ideas

Never lose a video idea again. Capture everything, prioritise the best ones with SEO scoring.

### Database: Ideas

Import `ideas.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Title | Title |
| Category | Select: Tutorial / Review / Vlog / Listicle / Commentary / Behind the Scenes / Collaboration / Shorts |
| Priority | Select: Hot / Medium / Low / Someday |
| SEO Potential | Select: High / Medium / Low |
| Competition | Select: High / Medium / Low |
| Search Volume | Select: High / Medium / Low |
| Target Audience | Text |
| Notes | Text |

### Idea Priorities

| Priority | Action |
|---|---|
| **Hot** | Script this week -- timely or audience-requested |
| **Medium** | Schedule within 2-3 weeks |
| **Low** | Backlog -- create when you have capacity |
| **Someday** | Review monthly, might never happen |

### Quick SEO Check

Before committing to a video: search the topic on YouTube. Are the top results from huge channels with millions of views? If so, competition is high. Look for topics where existing videos are outdated, poorly made, or missing a key angle you can provide.

### Recommended Views

- **Board View** -- Kanban by Priority
- **High SEO Potential** -- Filtered to SEO Potential = High
- **By Category** -- Grouped by Category

## Playlists

Organise your content into playlists that drive session watch time and help new viewers binge your channel.

### Database: Playlists

Import `playlists.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Playlist Name | Title |
| Videos Count | Number |
| Total Views | Number |
| Category | Text |
| Status | Select: Active / Growing / Archived |
| Strategy | Select: Evergreen / Series / Seasonal / Topical |
| Last Updated | Date |
| Notes | Text |

### Playlist Strategy

| Strategy | Purpose |
|---|---|
| **Evergreen** | Always relevant content grouped by theme -- drives search traffic |
| **Series** | Sequential videos that encourage binge watching |
| **Seasonal** | Time-sensitive compilations updated annually |
| **Topical** | Grouped by topic for browse and search discovery |

### Recommended Views

- **All Playlists** -- Table sorted by Total Views descending
- **Active Only** -- Filtered to Status = Active or Growing

### Playlist Tip

Put your best-performing video first in every playlist. Use keyword-rich playlist titles -- they rank in YouTube search just like videos do.
