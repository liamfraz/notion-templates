# Mockup Image Requirements — Etsy Listings

Etsy allows up to 10 images per listing. Aim for 7-10 per template. First image is the thumbnail — it must sell the product in search results.

**Image Specs:** 2700x2025px (landscape, 4:3 ratio) recommended. Minimum 2000px on shortest side. PNG or JPG.

---

## Universal Images (use across all listings)

These 2 images appear in every listing:

1. **"How to Use" slide** — 3-step visual: Purchase → Import to Notion → Start using. Shows it's a digital download, not physical
2. **"What's Included" comparison** — Lite vs Premium feature table. Clean, readable. Drives upgrade from free version

## Thumbnail Design Guidelines

The thumbnail is the most important image. It determines click-through rate from search.

- **Text overlay:** Template name + one-line value prop (e.g., "Track AI Sessions. Monitor Costs. Ship Faster.")
- **Background:** Clean mockup of the dashboard (not a raw screenshot — add slight perspective, shadow, or device frame)
- **Brand:** FlowTech logo or wordmark in corner. Consistent across all 5 listings
- **Colours:** Use a consistent palette across all listings. Dark backgrounds perform well for professional/tech products
- **Font:** Sans-serif, bold. Readable at thumbnail size (search results show images ~300px wide)
- **Avoid:** Cluttered screenshots, tiny text, too many colours, stock photos of people

## Image Creation Tools

- **Canva** — fastest for mockup slides with text overlays. Use "Digital Product Mockup" templates
- **Figma** — best for consistent branded mockups if you want more control
- **Screenshots** — take from actual Notion workspace with sample data loaded. Crop to focus area
- **Device frames** — wrap screenshots in laptop/tablet frames for professional look (Canva has these built in)
