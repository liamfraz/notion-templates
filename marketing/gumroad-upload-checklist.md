## Per-Product Upload Steps

Repeat for each of the 10 products:

### 1. Create Product
1. Go to gumroad.com/products → **New Product**
2. Product type: **Digital product**
3. Name: Copy from `gumroad-listings.md` → **Product Name**
4. Price: Set currency to **AUD** (except bundle). Enable "Pay what you want" where listed
5. Upload file: The premium ZIP from `/zips/`

### 2. Add Content
6. **Short description**: Copy from `gumroad-listings.md`
7. **Full description**: Copy the Long Description + Competitor Comparison + FAQ sections
8. Add the cross-sell snippet at the bottom (update `#` with actual bundle URL after bundle is live)

### 3. Images & Tags
9. Upload screenshots (6-8 per product — see image suggestions in listings file)
10. Set cover image to hero shot
11. **Tags**: Copy from `gumroad-listings.md` (8 tags per product, all include `notion template` + `2026`)
12. **URL slug**: Copy from `gumroad-listings.md`

### 4. Settings
13. Enable "30-day money-back guarantee" in refund policy
14. Category: **Templates**
15. Publish
