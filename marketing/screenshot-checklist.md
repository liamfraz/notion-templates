# Screenshot Checklist — Gumroad Listings

Take these screenshots BEFORE listing each product. Gumroad allows up to 7 images per product. Aim for 5-6 per listing.

**General rules:**
- Use a clean Notion workspace (no personal data visible)
- Dark mode OFF (light mode screenshots convert better on Gumroad)
- Browser width: 1440px (standard laptop)
- Crop to just the Notion content area (no browser chrome)
- Save as PNG, 1920x1080 or 2:1 ratio where possible
- Add the product name as a text overlay on image 1 (hero shot)

## Screenshot Tools

- **macOS:** Cmd+Shift+4 then Space to capture a window, or use CleanShot X
- **Overlay text:** Canva (free) or Figma — use DM Serif Display font to match the showcase site
- **Batch resize:** `sips -Z 1920 *.png` in Terminal to max-dimension resize
