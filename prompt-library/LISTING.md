### What is the AI Prompt Library & Workflow Manager?

A Notion template for anyone who uses AI tools regularly and wants to stop wasting time rewriting prompts. Store every prompt you use, rate the outputs, chain prompts into repeatable workflows, and track which ones actually save you time.

It's not a list of generic prompts copied from the internet. It's a system for building, managing, and improving your own prompt library -- the prompts that work for your specific work, tested and versioned over time.

### What's Inside

**Prompts Database (50 pre-built prompts)**
Genuinely useful prompts across 8 categories: Writing, Coding, Marketing, Analysis, Research, Creative, Business, and Productivity. Each prompt uses the CRAFT framework (Context, Role, Action, Format, Tone) and includes model compatibility notes, ratings, version tracking, and usage history.

**Workflows Database (12 pre-built workflows)**
Chain prompts together into repeatable multi-step processes. Content Pipeline, Code Review Cycle, Market Research Sprint, Client Proposal Builder, and more. Each workflow tracks steps, success rate, and time saved.

**Collections Database (8 curated sets)**
Prompt playlists for specific jobs: Newsletter Writer Kit, Code Quality Suite, Marketing Starter Pack, Product Launch Kit, and more. Grab a collection when you need a complete toolkit for a task.

**Usage Log (20 sample entries)**
Track every prompt use with ratings, output quality, time saved, and notes. See which prompts and models perform best over time. Make data-driven decisions about what to keep and what to rewrite.

**Prompt Crafting Guide**
The complete CRAFT framework for writing effective prompts. Includes good vs bad examples, an iteration process, and common mistakes to avoid. Stop guessing and start crafting.

**Analytics & Monthly Review**
Monitor prompt performance, spot weak categories, and maintain your library with a 15-minute monthly checklist. Keep your library sharp, not stale.
