## Content Pipeline

Track every short-form video from idea to posted across TikTok, Instagram Reels, and YouTube Shorts.

### Database: Content Pipeline

Import `content-pipeline.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Title | Title |
| Platform | Select: TikTok / Instagram Reels / YouTube Shorts / All |
| Status | Select: Idea / Scripting / Filming / Editing / Scheduled / Posted / Viral / Archived |
| Category | Select: Tutorial / Trend / Story Time / POV / Get Ready With Me / Day in Life / Reaction / Educational / Comedy |
| Hook Type | Select: Question / Bold Claim / Controversy / Curiosity Gap / Pattern Interrupt / Before-After |
| Post Date | Date |
| Duration (seconds) | Number |
| Sound | Text |
| Notes | Text |

### Content Lifecycle

```
Idea -> Scripting -> Filming -> Editing -> Scheduled -> Posted -> Viral -> Archived
```

### Recommended Views

- **Board View** — Kanban by Status (drag videos through stages)
- **By Platform** — Table grouped by Platform
- **This Week** — Table filtered to current week
- **Filming Queue** — Filtered to Scripting and Filming status

### Batching Tip

Film 3-5 videos in one session. Batch your filming days (Tuesday/Thursday) and editing days (Wednesday/Friday). Schedule posts across the week using your Posting Schedule database.

## Hooks Library

Your hook is the first 1-3 seconds. It determines whether someone watches or scrolls. Build a library of proven hooks.

### Database: Hooks Library

Import `hooks-library.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Hook Text | Title |
| Category | Select: Question / Bold Claim / Controversy / Curiosity Gap / Pattern Interrupt / Before-After / How To / List |
| Niche | Select: Productivity / Tech / Lifestyle / Business / Finance / Fitness / Food / Travel |
| Times Used | Number |
| Avg Completion Rate (%) | Number (percent format) |
| Performance Rating | Select: Viral / Strong / Average / Weak |
| Notes | Text |

### Hook Types Quick Reference

| Type | Example |
|---|---|
| **Question** | "Why is nobody talking about this?" |
| **Bold Claim** | "I wasted 3 years doing this wrong" |
| **Curiosity Gap** | "What happens when you do this for 30 days" |
| **Pattern Interrupt** | Unexpected visual or audio in first second |
| **Before-After** | "Here's what changed in 30 days" |

### Recommended Views

- **All Hooks** — Table sorted by Performance Rating
- **Top Performers** — Filtered to Viral and Strong
- **By Category** — Grouped by hook type

## Posting Schedule

Consistency beats virality. Know when to post and stick to it.

### Database: Posting Schedule

Import `posting-schedule.csv` or create a table with these columns:

| Column | Type |
|---|---|
| Day | Title |
| Time Slot | Text |
| Platform | Select: TikTok / Instagram Reels / YouTube Shorts |
| Content Type | Select: Tutorial / Trend / Story Time / POV / Educational / Comedy |
| Status | Select: Scheduled / Posted / Skipped |
| Performance Rating | Select: Viral / Strong / Average / Weak |
| Notes | Text |

### Optimal Posting Times

| Platform | Best Times | Peak Days |
|---|---|---|
| **TikTok** | 7am, 12pm, 7-9pm | Tue, Thu, Sat |
| **Instagram Reels** | 7-9am, 5-7pm | Mon, Wed, Fri |
| **YouTube Shorts** | 12-2pm, 6-8pm | Tue, Thu, Sat |

### Recommended Views

- **Weekly Grid** — Table grouped by Day
- **By Platform** — Grouped by Platform
- **Active Schedule** — Filtered to Scheduled status

## Weekly Checklist

- [ ] Review Content Pipeline — are this week's videos on track?
- [ ] Move completed videos through statuses (Editing -> Scheduled -> Posted)
- [ ] Capture 3-5 new video ideas in the Pipeline
- [ ] Log hook performance for posted videos in the Hooks Library
- [ ] Check posting schedule adherence — did you hit your targets?
- [ ] Plan next week's content (platforms, hooks, time slots)
